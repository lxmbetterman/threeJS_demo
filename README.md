# 基于Webpack2 multiple entry的three.js的例子

基于 webpack2 实现的多入口项目脚手架。

### 安装

```
npm install
```

### 开发

```
npm start
```
> http://localhost:8080/view


### 构建

| 命令              | 说明           |
| --------------- | ------------ |
| `npm run dev`   | 开发环境构建，不压缩代码 |
| `npm run build` | 生产环境构建，压缩代码  |
